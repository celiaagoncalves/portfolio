# Célia Gonçalves — Portfolio

Personal portfolio website for Célia Gonçalves, Senior Product Owner & Test Lead at Critical Software.

## About

Static site built with vanilla HTML5, CSS3, and JavaScript — no frameworks or dependencies beyond Font Awesome icons.

**Live:** [celiaagoncalves.github.io/portfolio](https://celiaagoncalves.github.io/portfolio) *(or your custom domain)*

## Features

- Bilingual (EN / PT) with localStorage persistence
- Responsive layout (mobile-first)
- Animated skill bars on scroll (IntersectionObserver)
- Experience timeline, certifications, and language proficiency sections
- SVG favicon

## Stack

| Layer | Tech |
|---|---|
| Markup | HTML5 (semantic) |
| Styling | CSS3 (custom properties, Grid, Flexbox) |
| Behaviour | Vanilla JS (ES6+) |
| Icons | Font Awesome 4 (self-hosted) |
